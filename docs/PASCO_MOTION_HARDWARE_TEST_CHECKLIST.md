# PASCO PS-3219 하드웨어 점검표 — 등속 운동 그래프 앱

> 실제 **PASCO Wireless Motion Sensor PS-3219**로 앱을 검증할 때 사용하는 점검표입니다.
> 한 항목씩 확인하고, 문제가 생기면 교사용 **PASCO 운동 센서 연결 진단** 패널에서
> 진단 JSON을 저장해 주세요.

## 준비
- [ ] 1. Chrome 또는 Edge에서 **HTTPS 배포 페이지** 열기 (예: `https://pasco01.labbitory.com`)
- [ ] 2. PASCO **PS-3219 전원 켜기** (LED 확인)

## 연결
- [ ] 3. **`센서 연결하기`** 누르기
- [ ] 4. 블루투스 창에서 **Motion 센서 선택하기** (이름에 `Motion` 또는 6자리 식별번호)
- [ ] 5. 라이브 카드의 **`센서로부터의 거리`** 값이 표시되는지 확인
- [ ] 6. 손이나 판을 천천히 움직여 **거리 변화** 확인하기
- [ ] 7. **`현재 속력`** 값이 바뀌는지 확인하기
      - 속력이 안 나오면, 진단 패널의 "속력 자료 출처"가 `위치 자료로부터 계산한 값`인지 확인
- [ ] (검증) `연결 완료`는 **실제 위치값을 받은 뒤**에만 떠야 합니다.
      만약 `센서에는 연결했지만 위치 또는 속력 자료를 해석하지 못했습니다…`가 뜨면,
      진단의 **최근 원시 패킷(hex)** 과 측정 항목을 저장해 채널 레이아웃을 확정하세요.

## 측정
- [ ] 8. 자동차를 센서에서 **20 cm 이상** 떨어진 위치에 두기
- [ ] 9. `측정 준비` 후 자동차를 **센서에서 멀어지도록** 출발시키기
- [ ] 10. 운동이 감지되면 **자동 측정 시작** 확인하기 (`측정 중…`)
- [ ] 11. **5초 후 자동 종료** 확인하기
- [ ] 12. **시간–이동 거리 그래프** 확인하기 (거의 직선, (0,0)에서 시작)
- [ ] 13. **시간–속력 그래프** 확인하기 (0보다 큰 값에서 거의 수평, 음수 없음)
- [ ] 14. **1초 간격 표**(시간–이동 거리, 구간 거리/평균 속력) 확인하기

## 비교 / 분석 / 보고서
- [ ] 15. **두 번째 속력**으로 다시 측정하기
- [ ] 16. **두 측정 그래프 비교**하기 (더 빠른 쪽이 더 가파른 거리 그래프 + 더 높은 속력선)
- [ ] 17. 4단계에서 **탐구 질문 작성**하기
- [ ] 18. 5단계에서 **Gemini 피드백** 받기 (개인정보 미전송 확인)
- [ ] 19. 6단계에서 **보고서 생성**하기 (두 그래프 모두 포함되는지 확인)
- [ ] 20. **인쇄 또는 PDF 저장**하기 (그래프가 페이지에서 잘리지 않는지)

## 견고성
- [ ] 21. **연결 해제 후 다시 연결**하기
- [ ] 22. **새로고침 후 재연결**하기
- [ ] 23. 오류 발생 시 **진단 JSON 저장**하기 (`진단 정보 JSON 저장` 버튼)

> 비고: 실제 PS-3219로 위 항목을 통과해야 "실측 검증 완료"라고 말할 수 있습니다.
